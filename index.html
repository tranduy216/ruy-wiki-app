📡 Bài viết lưu trực tiếp trên GitHub
Mỗi bài viết = 1 GitHub Issue trong repository này.
Mọi người truy cập web đều thấy bài viết ngay lập tức — không cần backend!
Cần GitHub PAT (Personal Access Token) để viết/xóa bài:
  1. Truy cập github.com → Settings → Tokens
  2. Chọn Generate new token (classic)
  3. Tích vào quyền public_repo → nhấn Generate
  4. Copy token dán vào bên dưới
Token chỉ lưu trong trình duyệt này, không gửi đến đâu ngoài GitHub API.
Vui lòng nhập token!
Token hiện tại đã được lưu.
